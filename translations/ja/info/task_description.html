文字列の中から、最初の単語を取り出しましょう。

これは、First Word ミッションの簡易版です。

  • 文字列の先頭と末尾にスペースが入ることはありません。
  • 入力される文字列は、英字とスペースのみで構成されます。

入力: 文字列

出力: 文字列

例題:

{% if interpreter.slug == "js-node" %} 
firstWord("Hello world") == "Hello"
{% else %} 
first_word("Hello world") == "Hello"
{% endif %}

どのように使われるか: コマンドラインの先頭の単語はコマンドです。

前提条件: テキストはa-z, A-Z とスペースを含むことができます。
