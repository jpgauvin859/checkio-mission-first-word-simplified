On vous donne une chaîne de caractères et vous devez trouver son premier mot.

Ceci est une version simplifiée de la mission First Word , qui peut être résolue plus tard.

  • La chaîne d'entrée se compose uniquement des lettres sans accent et d'espace.
  • Il n'y a pas d'espace au début et à la fin de la chaîne.

En entrée: Une chaîne de caractère.

En sortie: Une chaîne de caractère.

Exemple:

{% if interpreter.slug == "js-node" %} 
firstWord("Hello world") == "Hello"
{% else %} 
first_word("Hello world") == "Hello"
{% endif %}

Comment le code est utilisé: Le premier mot est une commande dans une ligne de commande.

Condition préalable: Le texte peut contenir a-z, A-Z et des espaces.
